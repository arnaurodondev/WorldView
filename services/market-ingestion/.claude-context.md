# Market Ingestion Service (S2) — Agent Context

**Port**: 8002 | **DB**: `ingestion_db` | **Status**: Mature

## Mission
Scheduled polling of EODHD, Alpha Vantage, Polygon, Yahoo Finance. Raw (bronze) + canonical (silver) storage to MinIO. Backfill orchestration and provider budget tracking.

## Domain Entities
IngestionTask, PollingPolicy, ProviderBudget, Watermark

## Kafka Topics
| Direction | Topic | Notes |
|-----------|-------|-------|
| Produces | `market.dataset.fetched` | Claim-check pointer events (MinIO refs, not data) |

## Key Tables
ingestion_tasks, outbox_events, polling_policies, provider_budgets, watermarks

## Runtime Processes
API Server, Scheduler (APScheduler), Worker (task claims + execution), Outbox Dispatcher

## API Endpoints (6)
healthz, readyz, metrics, trigger, backfill, status, policies

## Pitfalls
- EODHD demo key only supports original 3 endpoints + limited earnings calendar
- Rate limit exceeded marked as `rate_limited`, not failed
- MinIO unavailable triggers RetryableError with backoff
- Negative cache (`neg:ing:{provider}:{symbol}`) prevents retry storms (120s TTL)
