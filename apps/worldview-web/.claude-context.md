# Worldview Web (Next.js 15) — Agent Context

**Port**: 3001 | **Framework**: Next.js 15 App Router | **Status**: Active development

## Mission
Professional-grade financial intelligence web terminal (Bloomberg/TradingView-tier).
Dark-only UI, shadcn/ui component library, IBM Plex fonts, Midnight Pro palette.

## Tech Stack
- Next.js 15.1.7 (App Router, React 19, Node SSR)
- shadcn/ui only (40+ Radix UI primitives + Tailwind CSS)
- TanStack Query v5 (server state)
- lightweight-charts 4.2.3 (OHLCV charts)
- react-grid-layout (workspace multi-panel)
- cmdk (command palette)
- pnpm 10+ (exact versions, 0 CVE audit)

## API Integration
- All calls via `/api/*` (Next.js rewrite → `API_GATEWAY_URL`)
- Gateway client: `lib/gateway.ts` (40+ typed methods)
- WebSocket: direct to S10 (`NEXT_PUBLIC_WS_BASE_URL`)
- SSE: chat streaming via EventSource

## Critical Rules
- **Frontend → S9 only** — never call backend services directly
- **shadcn/ui only** — no other component libraries
- **Dark mode permanent** — `class="dark"` on `<html>`, never light mode
- **IBM Plex Mono for ALL numbers** — highest-impact visual rule
- **pnpm exact versions** — no `^` or `~` prefixes
- **Heavy inline comments** — user is new to Next.js; explain WHY not just WHAT

## Key Directories
- `app/` — Next.js App Router pages
- `app/(app)/` — Protected route group (auth guard in layout)
- `components/ui/` — shadcn/ui auto-generated
- `components/shell/` — App-wide: Sidebar, TopBar, FlashOverlay, GlobalSearch
- `components/dashboard/` — Dashboard widgets
- `components/instrument/` — Instrument detail components
- `lib/gateway.ts` — Typed S9 API client
- `contexts/` — AuthContext, AlertStreamContext

## Pitfalls
- `next.config.ts` rewrites strip `/api` prefix: `/api/v1/X` → `API_GATEWAY_URL/v1/X`
- WebSocket needs `?token=` query param (browser WS API has no headers)
- `"use client"` directive required for any component using hooks, events, or browser APIs
- TanStack Query staleTime should be set per-query (not globally)
- `react-grid-layout` layout stored in localStorage for persistence
