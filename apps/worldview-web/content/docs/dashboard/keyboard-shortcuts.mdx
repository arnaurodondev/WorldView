Worldview is designed for keyboard-first navigation. The most common actions have chord shortcuts that let you navigate the entire terminal without reaching for the mouse.

## Navigation chords

Press `G` (for "Go") and then the section letter within 1 second:

| Chord | Destination |
|-------|------------|
| `G` then `D` | Dashboard |
| `G` then `P` | Portfolio |
| `G` then `S` | Screener |
| `G` then `A` | Alerts |
| `G` then `C` | Chat / AI |
| `G` then `W` | Workspace |
| `G` then `X` | Settings |

<Callout type="info" title="Chord timeout">
If you press `G` but do not follow with a second key within 1 second, the chord resets. This prevents accidental navigation when typing in a text field. Chords are disabled when a text input, textarea, or contenteditable element has focus.
</Callout>

## Command palette

Press `⌘K` (macOS) or `Ctrl+K` (Windows/Linux) to open the global command palette from anywhere in the app.

The palette supports:

- **Instrument search** — type a ticker or company name, press Enter to open the chart
- **Navigation commands** — type "Portfolio", "Screener", "Alerts" to jump directly
- **Action commands** — "New alert", "Add to watchlist", "Open chat"
- **Docs search** — type any term to search documentation pages

Use `↑` and `↓` to move through results, `Enter` to select, `Escape` to close.

## In-page shortcuts

| Shortcut | Action | Context |
|----------|--------|---------|
| `⌘K` / `Ctrl+K` | Open command palette | Any page |
| `Escape` | Close modal / panel | Modal open |
| `?` | Show shortcut reference | Any page |
| `R` | Refresh current widget data | Dashboard |
| `N` | Focus notification bell | Any page |

## Screener shortcuts

| Shortcut | Action |
|----------|--------|
| `⌘F` / `Ctrl+F` | Focus filter search |
| `⌘E` / `Ctrl+E` | Export visible rows to CSV |
| `⌘S` / `Ctrl+S` | Save current screen as preset |

## Chat shortcuts

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift+Enter` | New line in message |
| `↑` | Edit last message |
| `/` (at start) | Trigger slash command autocomplete |

## Cheat sheet

Press `?` anywhere in the app (when not in a text input) to open the keyboard shortcut cheat sheet overlay. It shows all active shortcuts for the current page context.

<Callout type="tip" title="Customize shortcuts">
Keyboard shortcuts are not currently user-customizable. If you have a specific binding request, submit it via the feedback button.
</Callout>
