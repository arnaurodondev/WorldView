PR-G adds the architecture test LAST so it can codify the steady state without blocking the migration itself. Per the user's "small focused diffs" rule (RULES.md R1), this is the only compliant path. | 17-18 dev-hours total + 1 day visual QA; same total as single-PR, distributed across 7 reviewable commits. | All 633 rounded sites + 158 transition + ~80 ring + ~40 shadow + 184 hover-bg = ~1,800 sites total across ~120 files.
