## BLOCKING Issues — RESOLVED

### F-001 — Missing auth guard on `GET /v1/articles/{id}/impact-history`
- **Severity**: BLOCKING
- **Confidence**: HIGH (QA + Security agents)
- **File**: `services/api-gateway/src/api_gateway/routes/content.py:268`
- **Issue**: Route called `_auth_headers()` and `_clients()` without first checking `request.state.user`. Any unauthenticated caller could proxy requests to the internal S6 endpoint, bypassing tenant scoping.
- **Fix applied**: Added `if not getattr(request.state, "user", None): raise HTTPException(status_code=401)` guard. Also changed `article_id: str` → `article_id: UUID` to prevent path-traversal via non-UUID strings.
- **Status**: FIXED in `b7b60382`

### F-002 — Return type mismatch in `get_portfolio_sector_attribution`
- **Severity**: BLOCKING
- **Confidence**: HIGH (Architecture agent)
- **File**: `services/api-gateway/src/api_gateway/routes/portfolio.py:790`
- **Issue**: Function declared `-> PortfolioSectorAttributionResponse` but returned `Response(...)` on error paths with `# type: ignore[return-value, no-any-return]`. FastAPI's response model validation could interfere with error passthrough.
- **Fix applied**: Changed return type annotation to `-> Any` (consistent with all other mixed-return routes in the codebase); removed `# type: ignore` comment.
- **Status**: FIXED in `b7b60382`

## MAJOR Issues

### F-003 — Yield-curve P2 trigger condition wrong (`all` → `any`)
- **Severity**: MAJOR
- **Confidence**: HIGH (DS + QA agents)
- **File**: `services/api-gateway/src/api_gateway/routes/market.py:1521`
- **Issue**: `if all(v is None for v in yield_by_maturity.values())` only triggers ETF fallback when ALL P1 maturities are absent. With partial P1 data (e.g. 2Y and 10Y available but 5Y and 30Y missing), the ETF proxy was never called, leaving gaps where values should have been filled.
- **Fix applied**: Changed to `any(v is None ...)` so the ETF proxy fills partial gaps.
- **Status**: FIXED in `b7b60382`

### F-004 — `source = "etf_proxy"` set outside `if mat:` block
- **Severity**: MAJOR
- **Confidence**: HIGH (DS + QA agents)
- **File**: `services/api-gateway/src/api_gateway/routes/market.py:1567`
- **Issue**: `source = "etf_proxy"` was indented at the snap loop level, not inside the `if mat:` block. This meant `source` was set to `"etf_proxy"` even when the ETF ticker-to-maturity mapping lookup failed (i.e. when no maturity was actually populated), causing incorrect `source` attribution in the response.
- **Fix applied**: Indented `source = "etf_proxy"` inside `if mat:`.
- **Status**: FIXED in `b7b60382`

### F-005 — Operator precedence in yield-curve title matching
- **Severity**: MAJOR
- **Confidence**: HIGH (DS + Architecture agents)
- **File**: `services/api-gateway/src/api_gateway/routes/market.py:1509`
- **Issue**: `if maturity in title and "YIELD" in title or "TREASURY" in title and maturity in title` has implicit precedence: `and` binds tighter than `or`, so the condition was evaluated as `(maturity in title and "YIELD" in title) or ("TREASURY" in title and maturity in title)`. This is technically correct but non-obvious and fragile. Added explicit parentheses for clarity and correctness guarantee.
- **Fix applied**: Added explicit parentheses.
- **Status**: FIXED in `b7b60382`

### F-006 — NL screener allowlist validation silently disabled when S3 unavailable
- **Severity**: MAJOR
- **Confidence**: MEDIUM (Security + Architecture agents)
- **File**: `services/api-gateway/src/api_gateway/routes/market.py` (nl_screener_translate)
- **Issue**: When `GET /v1/fundamentals/screen/fields` fails or returns an empty set, `valid_fields` is empty and the allowlist check `if valid_fields and not all(f in valid_fields ...)` short-circuits to True, letting any field name through.
- **Decision**: Deferred — acceptable degradation for MVP (S3 outage is already a catastrophic failure mode; blocking NL screener on top adds no user value). Added to backlog as tech debt.
- **Status**: DEFERRED

### F-007 — Sequential ETF ticker lookups (latency)
- **Severity**: MAJOR
- **Confidence**: MEDIUM (DS agent)
- **File**: `services/api-gateway/src/api_gateway/routes/market.py` (yield_curve, ETF proxy loop)
- **Issue**: 4 ETF ticker searches are performed sequentially in a for-loop (worst case ~150ms latency at 40ms/req). Should use `asyncio.gather()`.
- **Status**: Deferred to a follow-up performance pass (yield-curve is not on the critical path for page load).
