Motivation

The information gap

Institutional desk

  • Bloomberg Terminal — unified data + news + analytics
  • NLP pipelines — articles scored by market relevance the moment they publish
  • Knowledge graphs — entities linked across filings, news, and price events
  • RAG-powered research — ask questions, get answers grounded in data

Individual investor

  • Scattered tabs — Yahoo Finance, Seeking Alpha, Twitter, SEC EDGAR
  • No scoring — every headline looks equally important
  • No context — can't see how entities relate across events
  • $24k/year — Bloomberg entry price; effectively zero access

The tooling gap isn't about data availability — EODHD, Polymarket, and public filings make data cheap. It's about the intelligence layer on top of it.

Data layer

How Worldview uses EODHD

  • End-of-day prices + OHLCV
    Core time-series for every tracked instrument. Used to compute article impact windows (price Δ at T+0/1/2/5 days).
  • Fundamental data
    Income statement, balance sheet, cash flows per entity. Drives the Fundamentals tab in the entity detail panel.
  • News API
    Primary article source. Every article enters the NLP pipeline for entity extraction, embedding, and relevance scoring.
  • Insider transactions
    Form 4 filings. Surfaced as timeline events in the knowledge graph and flagged as high-signal inputs for alert scoring.
  • Economic indicators + calendar
    Macro events (CPI, FOMC, NFP) ingested as temporal nodes in the knowledge graph. Regional filtering supported.

Why EODHD specifically

Breadth across fundamentals, news, and historical prices from a single provider — without the five-figure enterprise contract. At thesis scale, that's what made a production-grade multi-source architecture actually feasible.

Technical design

Worldview is a 10-service event-driven monorepo. Each service owns its domain completely — no cross-service database access. All inter-service communication goes through Kafka or REST.

Core feature

The Knowledge Graph

The knowledge graph is the connective tissue of the platform — it turns isolated data points into a navigable web of context.

What nodes represent

  • Financial instruments — stocks, ETFs, bonds
  • Companies & organisations — the legal entities behind instruments
  • People — executives, analysts, regulators (extracted from news & filings)
  • Events — earnings, M&A, macro announcements, insider trades
  • Topics — macro themes (inflation, rate cuts, AI capex)

What edges represent

  • MENTIONED_IN — entity ↔ article co-occurrence
  • CORRELATED_WITH — price correlation across instruments
  • EMPLOYS / EMPLOYED_BY — company ↔ person relationships
  • IMPACTED_BY — instrument ↔ macro event with price delta

How it is built

Article ingested
→
S6 NLP pipeline runs GLiNER NER
↓
Entities resolved to canonical IDs
↓
BGE embeddings stored in pgvector
↓
Edges created/updated in S7 graph DB
↓
Graph queryable via S9

Why this matters

When Apple reports earnings, the graph already knows which suppliers are mentioned in analyst coverage, which executives filed recent insider trades, and which macro events historically correlated with its price movement. That context surfaces automatically — no manual curation required.

Relevance scoring pipeline

Every article receives a composite score:
display_score =
  0.5 × market_relevance
  + 0.4 × llm_relevance
  + 0.1 × routing_score

Impact windows

For each article, price impact is measured at T+0, T+1, T+2, and T+5 days across all mentioned instruments. This creates a labelled dataset linking news events to verifiable market outcomes.

LLM scoring model

Qwen2.5:3b running locally — title-only prompt, zero latency cost. Output: 0–1 relevance score per entity–article pair.

Interface — State F

RAG-powered intelligence chat

[ screenshot — RAG chat interface ]

How it works

  • User submits natural-language question
  • nomic-embed-text encodes the query
  • pgvector HNSW index retrieves top-k article chunks
  • bge-reranker-v2-m3 re-ranks results
  • DeepSeek R1 Distill 32B generates grounded answer

Example queries

"Why did NVDA drop 4% on Tuesday?"
"What are analysts saying about Apple's supply chain?"
"Which companies have high insider selling this month?"

Chat model

DeepSeek R1 Distill 32B via DeepInfra
Fallback chain: OpenRouter → Groq → Ollama local
