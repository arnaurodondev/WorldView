**DLQRepositoryPort interface** (add to `application/ports/repositories.py`):
```python
@dataclass
class DLQEntry:
    """Value object returned by DLQRepositoryPort ŌĆö no ORM dependency."""
    dlq_id: UUID
    original_event_id: UUID
    topic: str
    error_detail: str | None
    status: str
    payload_json: dict | None
    created_at: datetime
    resolved_at: datetime | None
    resolution_note: str | None

class DLQRepositoryPort(ABC):
    @abstractmethod
    async def list_open(
        self, limit: int, offset: int
    ) -> tuple[list[DLQEntry], int]: ...

    @abstractmethod
    async def get_by_id(self, dlq_id: UUID) -> DLQEntry | None: ...

    @abstractmethod
    async def requeue(self, dlq_id: UUID) -> UUID: ...       # returns new event_id

    @abstractmethod
    async def mark_resolved(self, dlq_id: UUID, note: str | None) -> bool: ...
    # returns False if dlq_id not found
```

**DLQAdminUseCase** (`application/use_cases/dlq_admin.py`):
```python
class DLQAdminUseCase:
    def __init__(self, repo: DLQRepositoryPort, session: AsyncSession) -> None:
        self._repo = repo
        self._session = session

    async def list_open(self, limit: int, offset: int) -> tuple[list[DLQEntry], int]:
        return await self._repo.list_open(limit=limit, offset=offset)

    async def get_entry(self, dlq_id: UUID) -> DLQEntry:
        entry = await self._repo.get_by_id(dlq_id)
        if entry is None:
            raise DLQEntryNotFoundError(dlq_id)
        return entry

    async def retry_entry(self, dlq_id: UUID) -> UUID:
        entry = await self._repo.get_by_id(dlq_id)
        if entry is None:
            raise DLQEntryNotFoundError(dlq_id)
        new_id = await self._repo.requeue(dlq_id)
        await self._session.commit()
        return new_id

    async def resolve_entry(self, dlq_id: UUID, note: str | None) -> None:
        found = await self._repo.mark_resolved(dlq_id, note)
        if not found:
            raise DLQEntryNotFoundError(dlq_id)
        await self._session.commit()
```

**DLQEntryNotFoundError**: Add to `domain/errors.py` (subclass of `DomainError`).

**Infrastructure change**: Update `infrastructure/db/repositories/dlq.py` ŌĆö make
`DLQRepository` implement `DLQRepositoryPort` (add type hint, ensure method signatures match).

**DI wiring** (add to `api/deps.py` or inline as FastAPI dependency):
```python
async def _get_dlq_use_case(session: DbSessionDep) -> DLQAdminUseCase:
    from content_store.infrastructure.db.repositories.dlq import DLQRepository
    return DLQAdminUseCase(repo=DLQRepository(session), session=session)

DLQUseCaseDep = Annotated[DLQAdminUseCase, Depends(_get_dlq_use_case)]
```

**Acceptance criteria**:
- [ ] `DLQRepositoryPort` is a pure ABC with no imports from `infrastructure/`
- [ ] `DLQAdminUseCase` imports only from `application/` and `domain/`
- [ ] `DLQRepository` implements all methods of `DLQRepositoryPort`
- [ ] mypy passes with no new errors

##### T-C-1-02: Create scripts/run-service-e2e.sh

**Type**: config
**depends_on**: none
**blocks**: none
**Target files**:

**What to build**:
A helper script that runs e2e tests for a single service with the correct
database URL env vars injected. The docker-compose test stack must already
be running on localhost:55433.

```bash
#!/usr/bin/env bash
# scripts/run-service-e2e.sh <service-name>
# Runs e2e tests for a single service with correct env vars.
# Requires docker-compose test stack running (make infra-up).
set -euo pipefail

SERVICE="${1:?Usage: $0 <service-name>}"
ROOT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"
PG_BASE="postgresql+asyncpg://postgres:postgres@localhost:55433"

case "$SERVICE" in
  content-ingestion)
    export CONTENT_INGESTION_E2E_DATABASE_URL="${PG_BASE}/content_ingestion_db"
    ;;
  content-store)
    export CONTENT_STORE_E2E_DATABASE_URL="${PG_BASE}/content_store_db"
    ;;
  nlp-pipeline)
    export NLP_PIPELINE_E2E_DATABASE_URL="${PG_BASE}/nlp_db"
    ;;
  knowledge-graph)
    export KNOWLEDGE_GRAPH_E2E_DATABASE_URL="${PG_BASE}/intelligence_db"
    ;;
  alert)
    export ALERT_E2E_DATABASE_URL="${PG_BASE}/alert_db"
    ;;
  *)
    echo "ERROR: Unknown service: $SERVICE" >&2
    exit 1
    ;;
esac

SERVICE_DIR="$ROOT_DIR/services/$SERVICE"
if [[ ! -d "$SERVICE_DIR/tests/e2e" ]]; then
  echo "No e2e tests found for $SERVICE" >&2
  exit 1
fi

echo "=== Running e2e tests for $SERVICE ==="
cd "$SERVICE_DIR"
python -m pytest tests/e2e/ -m e2e -v --tb=short
```

**Acceptance criteria**:
