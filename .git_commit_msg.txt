feat(all-services+infra): dual-mode env config and portfolio wave-02 completion

Dual-mode environment configuration for all 9 services:

  Mode 1 — Full Docker Compose (--profile runtime):
    cp services/<svc>/configs/docker.env.example services/<svc>/configs/docker.env
    Compose mounts docker.env; all hostnames are Docker-internal.

  Mode 2 — FastAPI dev server (make run) with infra from Compose:
    cp services/<svc>/configs/dev.local.env.example services/<svc>/configs/dev.local.env
    pydantic-settings loads the file automatically; all URLs point to localhost.
